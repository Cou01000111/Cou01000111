## Minecraft Widgets Gen

minecraftのリソースパックにあるホットバーにショートカットキーを付与して出力する。設定によっては元の画像が消えてしまうので注意。ホットバーのキー配列はゲームディレクトリにある設定ファイルを参照しているので、ホットバーのキーがおかしいときは別のゲームディレクトリを参照してみてください。対応minecraftバージョンは1.16.5のみです
